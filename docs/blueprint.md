# **App Name**: Pedidos Socliquei

## Core Features:

- Data Parsing: Parse data from the Google Sheets TSV format, splitting it into order ID and product information fields.
- Order ID Extraction: Extract and prominently display the order ID from the parsed data.
- Product Info Extraction: Extract the product information string from the parsed data.
- Detailed View: On click of the order ID, present product information, broken down and formatted for readability (name, variation, price, quantity, etc.).
- Mobile Responsive Layout: Design a responsive layout that adapts to different screen sizes for optimal viewing on mobile devices.
- External Data Loading: Fetch data from the provided Google Sheets URL: https://docs.google.com/spreadsheets/d/e/2PACX-1vQk1dhpyMwmAZW1u20d0bal7Mk3vT0r5wD6YconmF7MBZ965mrg9cebc3bdZI_ZSQDJmsNIFd6KVAxI/pub?gid=0&single=true&output=tsv

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to create a sense of trust and stability.
- Background color: Light gray (#EEEEEE) for a clean, modern look.
- Accent color: A vibrant purple (#BA68C8) to highlight interactive elements.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text for a modern, readable appearance.
- Use a card-based layout to display each order, making it easy to scan on mobile devices.
- Use clear and simple icons to represent different aspects of the order details (e.g., product image, quantity).
- Subtle animations to provide feedback when a user interacts with the order items.
